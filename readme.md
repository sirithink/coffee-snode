#简介
这是一个nodejs的博客

采用 `Express.js 3.0` mvc框架`MySQL`数据库和`Jade`模版引擎

使用CoffeeScript编写，部署于Appfog!

#build and run
将CoffeeScript编译成js

打开命令行build: 
```
build.bat
```

打开另一个run:

```
strat.bat
```

请删除掉views/common/_layout.jade中的百度统计代码！

# 代码清理
使用`clear.bat`会在项目同级目录生成一个`temp`目录，

`temp`中的代码用来上传到Appfog！
```
clear.bat
```
